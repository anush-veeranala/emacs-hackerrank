Emacs HackerRank
----------------

###### Emacs plugin for HackerRank code compile and basic test

Add

<code>
(load "path to hackerrank.el")
</code>

to your .emacs file
Restart emacs


On the top of your code, Add comment in this format

<code> contest: contest-name; challenge: challenge-name; </code>

shortcut f7
